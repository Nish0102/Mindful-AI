# 🧠 Mental Health Support Website

A responsive website with an AI-driven chatbot that offers guidance and resources on mental health topics. Built with Python and web technologies, the system aims to promote mental wellness through supportive dialogue and easy access to information.

---

## 🌟 Features

- 💬 Interactive chatbot to simulate supportive conversations
- 🧾 Data-driven responses powered by a structured `.csv` file
- 🌐 Clean and responsive web UI with `index.html`
- 🧠 Focused on empathy, awareness, and accessibility

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS (inside `index.html`)
- **Backend:** Python (`app.py`)
- **Data Source:** CSV-based Q&A logic (`data.csv`)
